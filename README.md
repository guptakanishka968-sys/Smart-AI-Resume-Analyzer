# Smart AI Resume Analyzer

An AI-powered Resume Analyzer built using Python and Streamlit that helps users analyze resumes,
identify missing skills, get ATS-based insights, and explore trending industry skills.


## Features

 Resume Analysis
 ATS Score Checking
 Skills Extraction
 Missing Skills Recommendation
 Trending Skills Suggestions
 Word Cloud Visualization
 Career AI Chat
 Interactive Streamlit UI

## Tech Stack

 Python
 Streamlit
 NLP
 Pandas
